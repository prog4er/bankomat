
Структура программного проекта
bankomat программы должны быть организованы в соответствии с модульным принципом, обеспечивающим разделение интерфейса, бизнес-логики, работы с данными и модуля машинного обучения. В корневом каталоге проекта располагаются основные файлы запуска: main.py — серверная часть приложения, ui.py — пользовательский интерфейс, requirements.txt — перечень зависимостей. В отдельных подкаталогах размещаются функциональные модули: папка core содержит основную логику работы банковского терминала: logic.py, папка ml — файлы обучения и применения моделей машинного обучения:__init__.py , infer.py , train.py , папка storage — SQL-схему базы данных: schema.sql .
